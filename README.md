## What this is

A synthesis of cognitive science, systems thinking, AI literacy, contemplative practice, and lived experience. Core concepts include:

- **Memnetic** — an original coined term for the point where transmitted ideas become stored identity
- **NLP²** — a framework for understanding language as a programming medium at computational, cognitive, and cultural scales simultaneously
- **Epistemic Hygiene** — the discipline of knowing how you know what you know
- **Intentional AI use** — AI as cognitive scaffold, not oracle
- **Sovereignty as an architectural principle** — not political, structural

---

## What it claims

Some things here are well-supported by research. Some are hypotheses under investigation. Some are philosophical positions developed through personal practice. This framework holds epistemic hygiene as a core value — that standard applies to itself. Read critically.

## A note on personal work

AI-assisted reflection has real potential for helping people work through old patterns. It also has real limits. Anyone working through serious trauma should have professional support alongside any self-directed practice. This framework is a tool, not a treatment.

---

## Vision

Potential future applications include therapeutic tools developed alongside medical professionals, AI literacy curriculum, and interactive goal-setting tools. The core framework is what exists now. Everything else is being built.
